Methodology & Sources

WTW emission factors: These defaults are synchronised with the Python package model. Electric grid: WTT = 563.7 gCO₂eq/kWh (adjustable via panel). Gasoline: WTT 450.6 + TTW 2320.7 gCO₂eq/L. Diesel: WTT 503.1 + TTW 2480.3 gCO₂eq/L.

Embodied LCA (B1–B3): Values are defined in gCO₂eq/vkm and converted to gCO₂eq/pkm using passenger load. B1 = vehicle production. B2 = use-phase infrastructure and services. B3 = end-of-life disposal.

PHEV weighting: Car PHEV default EV share = 66%; Taxi PHEV = 36%; Bus PHEV = 0%. All shares adjustable via the control panel.
